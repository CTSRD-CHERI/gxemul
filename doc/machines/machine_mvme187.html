mvme187 machine

The mvme187 machine is a VMEbus based machine with a Motorola 88100 CPU.


Running OpenBSD/mvme88k

Start the emulator with the following command line: 

	gxemul -e mvme187 bsd.rd
where bsd.rd is an OpenBSD/mvme88k ramdisk kernel.
