GXemul: Release notes

Release notes for GXemul 0.5.0

Copyright (C) 2003-2008 Anders Gavare


GXemul is a framework for full-system computer architecture emulation.


The changes between release 0.4.6.x and 0.5.0 are:

  • GXemul 0.5.0 is a complete rewrite of the emulator, in C++. (Previous versions were written in C.)

  • A preliminary GTK+ GUI is now built-in.

  • There are now finally install and uninstall makefile targets, which should simplify installation.

To build GXemul, you may need the following:

  • The gtkmm libraries and header files (gtkmm-2.4 >= 2.12.1), for the GUI.
  • Doxygen, for building C++ class documentation.
Both gtkmm and Doxygen are optional, GXemul will build without them.


Super-short mini FAQ:

Q: Why C++? Why the rewrite/redesign?
A: I am more used to plain C, but I thought it would be interesting to try to use C++ for a realistically-sized project. Hello World is not realistically sized. I thought the emulator would be a good project to work on. A rewrite was needed anyway, since GXemul development had stagnated.


To build the emulator, run the configure script, and then run make. This should work on most Unix-like systems, with few or no modifications to the source code.
